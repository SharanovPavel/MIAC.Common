-------MIAC.Domain.Core.Interfaces v.1.0.0-------
-------------------------------------------------

Сборка содержит описание различных интерфейсов уровня ядра домена:

IHasIdentityField - обобщенный интерфейс, определяющий паттерн "Поле идентификации" (Identity field) (http://design-pattern.ru/patterns/identity-field.html)

IHasGlobalIdentificator - интерфейс, предоставляющий глобальный идентификатор (GUID)

IHasUpdateTime - интерфейс, предоставляющий свойство времени обновления состояния сущности

IHasCreationTime - интерфейс, предоставляющий свойство времени обновления состояния сущности

IHasTimeStamps - агрегирует указанные интерфейсы временных меток, определяя объект, имеющий временные все перечисленные выше временные метки
