-------MIAC.Patterns.Creational v.1.0.0-------
----------------------------------------------

Содержит интерфейсы и абстракции порождающих паттернов проектирования (creational patterns):

	- ICanCreateObjects - обобщенный интерфейс для большинства порождающих паттернов

	- паттерн "Строитель" (Builder) 

		- IBuilder - определяет базовый интерфейс паттерна "Строитель" (Builder)

		- BuilderBase - представляет абстрактный строитель объектов

	- паттерн "Фабричный метод" (Factory Method)

		- ICreator - представляет обобщенный интерфейс для реализации паттерна "Фабричный метод"
